Dein Name und deine Einstellungen werden lokal in deinem Browser gespeichert. Die Sticker-Daten werden in deiner Firebase-Datenbank geteilt.

📖 Schritt-für-Schritt Anleitung (Firebase + GitHub Pages)

1. Firebase Datenbank einrichten (kostenlos):
→ console.firebase.google.com öffnen
→ "Neues Projekt erstellen" → Projekt-Name eingeben
→ Links: "Realtime Database" → "Datenbank erstellen" → Testmodus wählen
→ Die URL oben (z.B. https://xxx-rtdb.firebaseio.com) hier eintragen

2. App auf GitHub Pages hosten:
→ GitHub-Account erstellen → "New Repository" → Name: panini-wm2026
→ Diese HTML-Datei hochladen (oder per git push)
→ Settings → Pages → Branch: main → Save
→ Deine URL: https://deinname.github.io/panini-wm2026
→ Diese URL mit Freunden teilen — fertig!

3. Claude API Key (für Foto-Erkennung):
→ console.anthropic.com → API Keys → Create Key
→ Key beginnt mit sk-ant-...

📷 Foto-Erkennung mit KI

Fotografiere eine Seite deines Heftes und lade das Bild hoch. Die KI erkennt automatisch, welche Sticker eingeklebt sind (ausgefüllt) und welche noch fehlen (leer).

ℹ️ Tipps für gute Ergebnisse

• Fotografiere die Seite gerade von oben, ohne Schatten
• Beide Seiten des aufgeschlagenen Heftes nacheinander scannen
• Das Bild sollte scharf und gut beleuchtet sein
• Die KI erkennt anhand der Stickernummern im Heft, welche Plätze leer oder gefüllt sind
• Nach dem Scan kannst du die Ergebnisse manuell korrigieren
